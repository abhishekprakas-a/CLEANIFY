# PWA Icons

Place the following PNG icons here (referenced by `src/app/manifest.ts`):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-maskable-512.png` — 512×512 (maskable, with safe-zone padding)

Generate them from your logo with any PWA asset tool, e.g.
[`@vite-pwa/assets-generator`](https://vite-pwa-org.netlify.app/assets-generator/)
or [realfavicongenerator.net](https://realfavicongenerator.net/).
